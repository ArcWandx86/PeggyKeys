# PeggyKeys
Break some keys? Here's help.

## Mappings
| Key | Remapping         |
|-----|-------------------|
| `l` | `alt`+`1`         |
| `L` | `alt`+`shift`+`1` |
| `v` | `alt`+`6`         |
| `V` | `alt`+`shift`+`6` |
| `o` | `alt`+`0`         |
| `O` | `alt`+`shift`+`0` |
| `.` | `alt`+`,`         |
| `>` | `alt`+`shift`+`,` |
| `9` | `alt`+`8`         |
| `(` | `alt`+`shift`+`8` |
